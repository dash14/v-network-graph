v-network-graph

An interactive network graph visualization component for Vue 3

Getting Started
Examples

🖌 SVG based

Draws a graph by SVG elements based on your data reactively.

👆 Interactive

Supports user interactions such as pan, zoom, drag nodes, and select, also with multi-touch.

🛠 Fully configurable

You can change the appearance and behavior of the interaction by the configuration.
